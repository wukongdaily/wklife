# 在Linux上快速部署一些好用的docker项目。

## Armbian/Ubuntu/Debian/Synology 7.2等基于Debian的Linux
```bash
wget -qO pi.sh https://cafe.cpolar.cn/wkdaily/zero3/raw/branch/main/zero3/pi.sh && chmod +x pi.sh && ./pi.sh
```
